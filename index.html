Welcome to FSAnalyzer

This is a simple tool I write to list all the files existed in a specific file system in order to track the timeline of some files you we may concern. This is a project for learning. The implementation It was in the summer of 2015 that I take a lesson called Introduction to Digital Forensics and I tried to write my own analyze tool and here it is.

History

I first know about this kind of knowledge from a book named FSFA(File System Forensic Analysis) written by Brain Carrier. I tried his tool TSK(The Sleuth Kit) and it is wonderful.
